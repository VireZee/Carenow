# ExpressJS
`modify the db.js and knexfile.js on "server/db.js", "server/knexfile.js"`
```sh
npm i
```
or (if needed)
```sh
npm i -f
```
then
```sh
npx knex migrate:latest
```
then
```
npm start
```
or
```sh
nodemon server.js
```
